## Usage Guidelines

1. **Consistency:** All visual assets should use the same color palette and typography
2. **Quality:** Images should be high-resolution and professionally rendered
3. **Branding:** Include SleepSmarter.io logo subtly on all covers
4. **Accessibility:** Ensure sufficient contrast for text readability
5. **Versatility:** Create both light and dark mode variants if possible
