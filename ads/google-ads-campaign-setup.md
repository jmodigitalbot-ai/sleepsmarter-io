## Bidding Strategy Recommendations

### Phase 1: Learning (Days 1-14)
- **Strategy:** Manual CPC
- **Goal:** Gather conversion data
- **Actions:** 
  - Set conservative bids (50-75% of suggested bid)
  - Monitor search terms report daily
  - Add negative keywords aggressively
  - Pause underperforming keywords

### Phase 2: Optimization (Days 15-30)
- **Strategy:** Maximize conversions (target CPA)
- **Goal:** Scale winning keywords
- **Actions:**
  - Switch to automated bidding after 15 conversions
  - Set target CPA based on Phase 1 performance
  - Increase bids on top-performing keywords by 20%
  - Create ad schedule adjustments based on conversion times

### Phase 3: Scaling (Day 31+)
- **Strategy:** Target ROAS
- **Goal:** Maximize return on ad spend
- **Actions:**
  - Set target ROAS at 300% (3:1 return)
  - Expand to similar keywords
  - Test new ad copy variations
  - Implement device bid adjustments
