## 4. Submission Strategy & Notes

### General Submission Guidelines:
1. **Website Readiness:** Ensure sleepsmarter.io is fully functional with clear value proposition
2. **Description:** Prepare a compelling 150-200 word description highlighting:
   - Free sleep calculator tool
   - Sleep science blog
   - Launch date (Feb 2026)
   - Target audience (adults seeking better sleep)
3. **Categories:** Use appropriate categories: Health > Sleep, Health > Tools, Wellness > Sleep, Calculators > Health
4. **Contact Info:** Use consistent email/contact information

### Directory Submission Priority:
**Tier 1 (High Authority):**
- Yahoo! Directory, BOTW, DMOZ, Business.com, AboutUs
- Submit within first month

**Tier 2 (Health Niche):**
- Health-specific directories (idmoz, Healthigg, Medical directories)
- Submit within first 2 months

**Tier 3 (General/Tools):**
- Tool directories, general web directories
- Ongoing submissions

### Resource Page Pitching:
1. **Research:** Find relevant "best sleep tools" articles
2. **Personalize:** Mention specific article and how sleepsmarter.io complements it
3. **Value:** Emphasize free tool, no registration required
4. **Follow-up:** Polite follow-up after 1-2 weeks

### Journalist Platform Strategy:
1. **Create Expert Profile:** Position as sleep health/technology expert
2. **Monitor Queries:** Set up alerts for sleep, health, wellness, technology queries
3. **Quick Responses:** Respond within 1-2 hours of query posting
4. **Quality Answers:** Provide substantive, quotable responses
5. **Include Link:** Naturally include sleepsmarter.io as resource

### Tracking & Monitoring:
- Use spreadsheet to track submissions
- Note approval/rejection dates
- Monitor backlink acquisition
- Update strategy based on results

### Estimated Timeline:
- **Month 1:** Tier 1 directories + major resource page pitches
- **Month 2:** Tier 2 directories + journalist platform setup
- **Month 3:** Tier 3 directories + ongoing pitching
- **Ongoing:** Monitor platforms, respond to queries, pitch new resource pages

### Success Metrics:
- Target: 50+ quality backlinks in first 3 months
- Focus on DR 30+ directories
- Prioritize dofollow links
- Track referral traffic from directories
