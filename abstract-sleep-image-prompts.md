## Tips for Using These Prompts:
- Always specify "abstract", "minimalist", "digital art", or "geometric" to avoid human figures
- Use color descriptions like "cool tones", "gradient blues", "soft pastels"
- Avoid words like "bed", "person", "sleeping", "body" - use conceptual terms instead
- Focus on shapes, patterns, colors, and textures that evoke sleep concepts
- Add style modifiers: "clean lines", "soft gradients", "ethereal", "minimalist design"
